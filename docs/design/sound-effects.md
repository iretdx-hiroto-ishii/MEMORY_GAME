## 2. 音源ファイル

| ファイル | ID | 用途 |
|----------|-----|------|
| `close.mp3` | `close` | ×マークなどの閉じるボタン押下 |
| `count.mp3` | `count` | 制限時間 5〜1 秒のカウントダウン時 |
| `finish.mp3` | `finish` | ゲーム終了時（タイムアップ含む） |
| `flip.mp3` | `flip` | カード選択時（1枚目のみ） |
| `miss.mp3` | `miss` | ペア不一致（2枚目をめくったタイミング） |
| `ok.mp3` | `ok` | 個別指定のないボタン押下 |
| `result.mp3` | `result` | ゲーム終了後、結果画面を初回表示するときのみ |
| `start.mp3` | `start` | ゲーム開始・リトライ時 |
| `success.mp3` | `success` | ペア一致（2枚目をめくったタイミング） |

---

## 3. 再生ルール

### 3.1 カード操作

| 操作 | 再生 |
|------|------|
| 1枚目をめくる | `flip` |
| 2枚目をめくり一致 | `success`（`flip` は鳴らさない） |
| 2枚目をめくり不一致 | `miss`（`flip` は鳴らさない） |

### 3.2 ゲーム進行

| タイミング | 再生 |
|------------|------|
| `resetGame()` 実行（スタート・リトライ） | `start` |
| 残り時間が 5, 4, 3, 2, 1 秒になったとき | `count` |
| クリアまたはタイムアップ確定 | `finish` |
| 上記終了後、結果 overlay を初回表示 | `result` |
| 🏁 などで 2 回目以降に結果 overlay を開く | なし |

### 3.3 UI 操作

| 操作 | 再生 |
|------|------|
| モーダルの × / 閉じる | `close` |
| 上記以外の汎用ボタン（メニュー項目・確認キャンセル・タイトル戻る等） | `ok` |
| サウンド設定をオンに切り替え | `ok` |
| サウンド設定をオフに切り替え | なし |
| 効果音スライダーを操作して手を離したとき | `ok`（音量プレビュー） |
| リトライ確定（`resetGame` へ） | `start`（`ok` は鳴らさない） |
